# Bayesian IRT Analysis Pipeline (Stan/cmdstanpy)

## Overview

Scripts **07–12** provide a full Bayesian counterpart to the MML pipeline (scripts 01–06) for the four Explanatory Item Response Models from Chapter 5 (Wilson, De Boeck & Carstensen, 2008).

Both pipelines share the same simulated data (generated by `01_generate_data.py`), so students can directly compare frequentist (MML) and Bayesian approaches.

## Installation

```bash
# 1. Install cmdstanpy and arviz
pip install cmdstanpy arviz matplotlib numpy pandas

# 2. Install CmdStan (C++ compiler required — one-time setup)
python -m cmdstanpy.install_cmdstan
```

## Pipeline

| Script | Corresponds to | What it does |
|--------|---------------|-------------|
| `01_generate_data.py` | — | Simulates data (shared by both pipelines) |
| `02_descriptive_analysis.py` | — | Exploratory analysis (shared) |
| `07_bayes_setup.py` | — | Checks installation, compiles all Stan models |
| `08_bayes_rasch.py` | `03_rasch_model.py` | Bayesian Rasch model |
| `09_bayes_latent_regression_rasch.py` | `04_latent_regression_rasch.py` | Bayesian person-explanatory model |
| `10_bayes_lltm.py` | `05_lltm.py` | Bayesian item-explanatory model |
| `11_bayes_latent_regression_lltm.py` | `06_latent_regression_lltm.py` | Bayesian doubly-explanatory model |
| `12_bayes_model_comparison.py` | (model comparison in 06) | LOO-CV / WAIC comparison + MML vs Bayesian |

## Run Order

```bash
python 01_generate_data.py          # generate data (if not done already)
python 07_bayes_setup.py            # compile Stan models
python 08_bayes_rasch.py            # Model 1
python 09_bayes_latent_regression_rasch.py  # Model 2
python 10_bayes_lltm.py             # Model 3
python 11_bayes_latent_regression_lltm.py   # Model 4
python 12_bayes_model_comparison.py # compare all
```

## Stan Models

Located in `stan_models/`:

- `rasch.stan` — η = θ_p − β_i
- `latent_regression_rasch.stan` — η = Σϑ_j Z_pj + θ_p − β_i
- `lltm.stan` — η = θ_p − Σβ_k X_ik
- `latent_regression_lltm.stan` — η = Σϑ_j Z_pj + θ_p − Σβ_k X_ik

All models use weakly informative priors: Normal(0,25) for fixed effects, Half-Cauchy(0,2.5) for scale parameters. All include `generated quantities` blocks for log-likelihood (needed for LOO-CV).

## What Students Learn

1. **Bayesian workflow**: specifying priors → MCMC sampling → convergence diagnostics → posterior summarization
2. **Uncertainty quantification**: posterior distributions and credible intervals (vs. point estimates + standard errors from MML)
3. **Model comparison**: LOO-CV / WAIC (vs. AIC / BIC)
4. **Comparing paradigms**: Bayesian posteriors should closely match MML point estimates for these well-identified models, demonstrating the equivalence for large samples

## Key Differences from MML Pipeline

| Aspect | MML (scripts 03–06) | Bayesian (scripts 08–11) |
|--------|---------------------|-------------------------|
| Estimation | Marginal ML + quadrature | Full MCMC (HMC/NUTS) |
| Output | Point estimates + SEs | Full posterior distributions |
| Item params | Fixed | Full posterior with CIs |
| Person params | EAP (post-hoc) | Jointly estimated |
| Model comparison | Deviance, AIC, BIC | LOO-CV (PSIS-LOO), WAIC |
| Uncertainty | Asymptotic SEs | Exact finite-sample posteriors |
| Computation | Fast (~seconds) | Slower (~minutes per model) |

## Troubleshooting

- **"CmdStan not found"**: Run `python -m cmdstanpy.install_cmdstan`
- **Divergent transitions**: Increase `adapt_delta` (e.g., 0.99)
- **Slow sampling**: Reduce data size in `01_generate_data.py` (change N to 200)
- **arviz not installed**: Script 12 falls back to a manual WAIC calculation
